clone或一个新的Interlij IDE项目到本地如何正常运行

# 步骤
* 设置src目录为Source Root
* 添加SDK
* 指定SDK和设置项目编译输出目录
* 这时候就可以选择.java文件点击 运行(Ctrl + Shift + F10)
