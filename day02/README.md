day02_03 赋值运算符与比较运算符的使用
==

# 赋值运算符
* 符号：=
>当"="两侧数据类型不一致时，可以使用自动类型转换或使用强制类型转换原则进行处理

> 支持连续赋值。

* 扩展赋值运算符：+=, -+, *=, /=, %=

# 比较运算符
|运算符 |运算 |示例 |结果 |
|:---|:---|:---|:---|
|== |相等于|4 == 3 |false |
|!= |不等于 |4 ~= 3 |true |
|< |小于 |4 < 3 |false |
|\> |大于 |4 > 3 |true |
|<= |小于等于 |4 <= 3 |false |
|>= |大于等于 |4 >= 3 |true |
|instanceof |检查是否是类的对象 |"Hello" is instanceof String |true |
